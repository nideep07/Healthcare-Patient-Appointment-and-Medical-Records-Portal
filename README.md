# Healthcare-Patient-Appointment-and-Medical-Records-Portal
An enterprise-grade, privacy-by-design healthcare web application built with Flask and MySQL. This platform bridges the gap between patient accessibility and clinical data security by enforcing strict, time-boxed data ownership models.

#The Core Philosophy
Traditional healthcare infrastructure leaves patient data fragmented across isolated clinic silos. This portal consolidates prescriptions, lab results, and diagnostic histories into a single chronological timeline. Instead of handing open-ended data access to providers, the system shifts ownership back to the patient via a 48-hour conditional consent registry—meaning practitioners can only view history when a patient explicitly approves a request or has a verified appointment scheduled for that exact day.
